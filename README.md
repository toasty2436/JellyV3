# HexaTV - Premium Streaming Platform

A sophisticated streaming platform .

## Key Features
- 🎬 100K+ movies and 70K+ tv episodes
- 🔖 Personal watchlist
- 🔍 search functionality
- 📱 Responsive design
- ✨ Premium UI/UX
- 🎨 Dynamic content loading

## Tech Stack
- React 17
- React Router DOM
- Tailwind CSS
- TMDB API
- Axios
- GitHub Pages
